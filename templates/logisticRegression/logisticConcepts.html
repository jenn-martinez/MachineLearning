Logistic Regression Concepts

What is logistic regression?

Logistic regression is a supervised learning algorithm used to solve classification problems. Unlike linear regression, it does not predict a continuous number but instead predicts the probability that a sample belongs to a specific class (0 or 1). It is characterized by:

  • It predicts probabilities between 0 and 1 using the sigmoid function.
  • It classifies data into discrete categories (Yes/No, Elite/Average).
  • It is used for spam detection, medical diagnosis, and player classification.

Types of classification

Binary classification

Binary classification predicts one of two possible outcomes. The model outputs a probability and then assigns a class based on a threshold (usually 0.5).

Example: Is a FIFA 24 player Elite (Overall ≥ 80)? → Yes (1) or No (0).

Multiclass classification

Multiclass classification predicts one of three or more possible outcomes. It extends binary logistic regression by using techniques like One-vs-Rest (OvR).

Example: Classifying a player as Bronze, Silver, or Gold tier.

Sigmoid function

The sigmoid function is the core of logistic regression. It transforms any real number into a value between 0 and 1, interpreted as a probability.

p = 1 / (1 + e-z)

Where z is the linear combination of inputs and weights. If p ≥ 0.5 → class 1 (Elite). If p < 0.5 → class 0 (Average).

Decision boundary

The decision boundary is the threshold that separates the two classes. It is the point where the model switches from predicting class 0 to class 1.

By default the threshold is 0.5, meaning if the predicted probability is greater than 50%, the model classifies the sample as class 1.

Model equation

For logistic regression, the model first computes a linear combination:

z = β₀ + β₁x₁ + β₂x₂ + ... + βₙxₙ

Then applies the sigmoid function to obtain the final probability:

p = 1 / (1 + e-z)

Where β₀ is the intercept and β₁...βₙ are the feature coefficients learned during training.

Assumptions about logistic regression

  1. The dependent variable must be binary or categorical
  2. The observations must be independent of each other
  3. There should be little or no multicollinearity among the input features
  4. A large sample size improves the reliability of the model

Example of application

A real-life example is classifying FIFA 24 players as Elite or Average. Where:

  • The independent variables are Pace, Shooting, Passing and Dribbling.
  • The dependent variable is whether the player is Elite (Overall ≥ 80)

The logistic regression model would be as follows:

z = β₀ + β₁(Pace) + β₂(Shooting) + β₃(Passing) + β₄(Dribbling)

p = 1 / (1 + e-z)

Where p is the probability of the player being Elite. If p ≥ 0.5 the model classifies the player as Elite, otherwise as Average.
