Logistic Regression Concepts

What is logistic regression?

Logistic regression is a supervised learning algorithm used to solve classification problems. It predicts the probability that a sample belongs to a specific class (0 or 1).

  • Predicts probabilities between 0 and 1 using the sigmoid function.
  • Classifies data into discrete categories (Yes/No, Elite/Average).
  • Used for spam detection, medical diagnosis, and player classification.

Types of Classification

Binary classification

Predicts one of two possible outcomes. Example: Is a FIFA 24 player Elite (Overall ≥ 80)? → Yes (1) or No (0).

Multiclass classification

Predicts one of three or more possible outcomes using One-vs-Rest (OvR). Example: Classifying a player as Bronze, Silver, or Gold.

Sigmoid Function

Transforms any real number into a value between 0 and 1:

p = 1 / (1 + e-z)

If p ≥ 0.5 → class 1. If p < 0.5 → class 0.

Decision Boundary

The default threshold is 0.5. If the predicted probability is greater than 50%, the model classifies the sample as class 1.

Model Equation

Linear combination:

z = β₀ + β₁x₁ + β₂x₂ + ... + βₙxₙ

Then applies sigmoid:

p = 1 / (1 + e-z)

Assumptions

  1. The dependent variable must be binary or categorical
  2. The observations must be independent of each other
  3. There should be little or no multicollinearity among input features
  4. A large sample size improves reliability
