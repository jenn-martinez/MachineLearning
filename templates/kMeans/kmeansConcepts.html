K-Means Clustering

Unsupervised Machine Learning Algorithm for Data Segmentation

What is K-Means Clustering?

K-Means is an unsupervised learning algorithm used to partition n observations into k clusters, where each observation belongs to the cluster with the nearest mean (centroid).

It is one of the most popular clustering algorithms due to its simplicity and efficiency for large datasets.

Key Idea: Similar data points are grouped together, and dissimilar points are separated.

Algorithm Steps

Step 1: Randomly select k points as initial centroids.
Step 2: Assign each data point to the nearest centroid.
Step 3: Recalculate centroids as the mean of all points in each cluster.
Step 4: Repeat steps 2-3 until convergence.

Mathematical Foundation

Objective Function (WCSS):

J = Σj=1k Σi=1n ||xi(j) - μj||²

Choosing the Number of Clusters (K)

Elbow Method

Plot WCSS against K values. The "elbow point" indicates optimal K.

Silhouette Method

Measures cluster quality. Values range from -1 to 1.

Distance Metrics

K-Means typically uses Euclidean distance:

d(x, y) = √( Σi=1n (xi - yi)² )
Important: Feature scaling is essential before applying K-Means.

Centroid Initialization

  • Random: Simple but can be suboptimal
  • K-Means++: Default in scikit-learn, better convergence

Evaluation Metrics

  • Inertia (WCSS): Lower is better
  • Silhouette Score: [-1,1], higher is better
  • Davies-Bouldin: Lower is better
  • Calinski-Harabasz: Higher is better

Pros & Cons

Advantages

  • Simple and fast
  • Scales well
  • Guaranteed convergence

Disadvantages

  • Need to specify K
  • Sensitive to initialization
  • Assumes spherical clusters

from sklearn.preprocessing import StandardScaler
