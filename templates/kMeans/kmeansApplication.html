K-Means Clustering Model

Customer Segmentation | Unsupervised Learning

Problem Description

The objective is to segment customers into distinct groups (clusters) based on their Annual Income and Spending Score. This unsupervised learning approach helps businesses understand customer behavior patterns and design targeted marketing strategies.

Dataset Explanation

The Customer Segmentation dataset contains {{ total_records }} records of retail customers. Each customer is described by two key behavioral features:

Variables used:
  • Annual Income (k$) - Customer's yearly earnings in thousands of dollars
  • Spending Score (1-100) - Composite score based on purchase frequency and amount
The dataset contains realistic customer behavior patterns commonly found in e-commerce businesses.

Data Preparation (Pipeline)

  • Cleaning: No missing values or duplicates detected
  • Scaling: StandardScaler (Z-score) applied - crucial for K-Means as it uses Euclidean distance
  • Transformation: Features transformed to mean=0, variance=1
  • Why scaling? Prevents features with larger scales from dominating distance calculations

Model Training (K-Means)

K-Means is an iterative algorithm that partitions data into K clusters by minimizing intra-cluster variance. The algorithm assigns each point to the nearest centroid, then recalculates centroids as cluster means.

Customer Segments Visualization

Scatter plot showing customer segments with cluster centroids (black X markers).

Hyperparameters:
  • Number of clusters (k): {{ n_clusters }}
  • Initialization: k-means++
  • Max iterations: 300
  • Random state: 42
  • n_init: 10
  • Distance metric: Euclidean distance
