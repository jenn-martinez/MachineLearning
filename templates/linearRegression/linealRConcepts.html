Linear Regression Concepts

What is linear regression?

Linear regression is a supervised learning algorithm used to model the relationship between a dependent variable and one or more independent variables.

  • It assumes a linear relationship between input and output.
  • It uses a line of best fit to make predictions.
  • It is used for forecasting, trend analysis, and predictive modeling.

Independent and Dependent Variables

Dependent variable

The dependent variable is the one whose outcome is conditioned by the independent variable. It is assigned the letter Y.

Independent variable

The independent variable is the data or factor used to make predictions. It is assigned the letter X.

Training Data

Training data is the information given to the machine learning model so that it learns to make predictions, recognize patterns, and generate content.

Regression Line

The regression line is the best-fit straight line that represents the relationship between variables. It minimizes the difference between actual values and predicted values.

Slope and Intercept

For simple linear regression:

y = mx + b

The slope (m) indicates how much Y changes when X increases by one unit. The intercept (b) is the value of Y when X = 0.

Assumptions about Linear Regression

  1. The relationship between X and Y is a straight line
  2. The errors do not affect each other
  3. The error should have the same dispersion among the data
  4. The errors must have a normal distribution

Gradient Descent

An optimization technique used to minimize prediction error by repeatedly adjusting parameters.

How it works

  • Assign random values for the slope and intercept
  • Calculate the error between predicted and actual values
  • Calculate how much each value contributes to the error
  • Update the parameters to reduce the error
  • Repeat until the error is minimized

Example of Application

A real-life example is advertising expenditure and revenue:

  • Independent variable: advertising investment
  • Dependent variable: revenue
