BBVA Pipeline

Predictive Model | Early Debt Resolution

Predictive model developed by BBVA AI Factory to improve early debt management and personalize strategies based on customer risk profiles.

Problem Description

Customers entering delinquency affect both themselves and the bank. The goal is early detection and classification to apply the best recovery strategy.

Type of Machine Learning

This is a Supervised Learning model using historical labeled data.

It performs classification by assigning customers to risk categories.

Classification & Models

Four models are used depending on delinquency status.

Data

Input Data

  • Sociodemographic data
  • Debt status
  • Payment behavior
  • Financial history

Output

  • Repayment probability
  • Risk classification
  • Recommended actions
