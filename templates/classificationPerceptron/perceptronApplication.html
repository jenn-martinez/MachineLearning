Perceptron classification model

Problem Description

The objective is to classify breast tumors as Benin (0) or Malignant (1) based on geometric characteristics obtained from digitized images of a fine needle aspiration (FNA) of a breast mass.

Dataset explanation

The Breast Cancer Wisconsin (Diagnostic) dataset, obtained from Kaggle, was used. This dataset contains: 30 tumor-related characteristics. For this model, the 10 most important average characteristics were selected. For practical purposes, the user is only asked for 4 physical dimensions; the remaining 6 are filled in with the average values for a normal person according to the original dataset. The 10 characteristics used are:

variables requested from the user:

  • Average radius
  • Average texture
  • Average perimeter
  • Average area

Filled variables:

Smoothness Compactness Concavity Concave points Symmetry Fractal Dim.

Preparation of the Data (Pipeline)

  • Cleaning: The IDs belonging to each diagnosis were removed and null values ​were used.
  • Scaling: Data standardization (Z-score) using the StandardScaler to transform the features into a distribution with mean mu=0 and variance sigma=1, crucial for Perceptron convergence.
  • Division: The dataset was divided into 80% for weighted training and 20% for testing.

Model training

The Perceptron learns by iteratively updating a vector of weights (W) and a bias (b). The activation function used is the Unit Step function.

Learning Curve (Loss)

It shows the reduction in updates (errors) as the model adjusts its weights at each epoch.

Hyperparameters:
  • Learning Rate: 0.01
  • Epoches: 100
  • Function: f(z) = 1 if z >= 0 else 0

Final Interpretation of the Results

The model exhibits robust performance. In the clinical setting, a high recall rate is the primary objective, as it indicates a low false negative rate (cases where cancer is present but was not detected).

Conclusion: Although the Perceptron is a model limited to linearly separable problems, the results demonstrate that, after adequate standardization of the variables, the dataset presents a structure that allows for effective classification with this simple architecture
