Perceptron Classification Model

Problem Description

The objective is to classify data into two categories using a perceptron model. This approach is commonly used in binary classification problems where the goal is to separate data using a linear decision boundary.

Dataset and Variables

The dataset consists of numerical input features that represent measurable characteristics. Each data point is labeled as either class 0 or class 1.

  • Input features (X)
  • Target labels (y)
  • Binary classification output

Model Training

  • Initialize weights and bias
  • Compute linear combination
  • Apply activation function
  • Update weights based on error
  • Repeat for multiple epochs

Prediction

After training, the perceptron can classify new input data by applying the learned weights and activation function to determine the output class.

Example Code (Expanded)

import numpy as np

class Perceptron:
    def __init__(self, learning_rate=0.1, epochs=100):
        self.lr = learning_rate
        self.epochs = epochs
        self.weights = None
        self.bias = None

    def fit(self, X, y):
        n_samples, n_features = X.shape
        self.weights = np.zeros(n_features)
        self.bias = 0

        for _ in range(self.epochs):
            for idx, xi in enumerate(X):
                linear_output = np.dot(xi, self.weights) + self.bias
                y_pred = 1 if linear_output >= 0 else 0

                update = self.lr * (y[idx] - y_pred)
                self.weights += update * xi
                self.bias += update

    def predict(self, X):
        linear_output = np.dot(X, self.weights) + self.bias
        return np.where(linear_output >= 0, 1, 0)


# Training data
X = np.array([
    [2, 1],
    [1, 1],
    [2, 0],
    [0, 1]
])

y = np.array([1, 1, 1, 0])

# Train model
model = Perceptron()
model.fit(X, y)

# Predictions
test_data = np.array([[1, 0], [0, 1], [2, 2]])

predictions = model.predict(test_data)

print("Predictions:", predictions)

Code Explanation

The provided code implements a basic Perceptron algorithm for binary classification. It defines a class that initializes the learning rate, number of epochs, weights, and bias.

During training, the model processes each data point, computes a linear combination, and applies a step activation function to determine the predicted class.

If the prediction is incorrect, the model updates its weights and bias using the error. This allows the algorithm to improve over multiple iterations.

Finally, the model is used to predict new data, demonstrating its ability to generalize from learned patterns.
