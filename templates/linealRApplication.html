Linear Regression - Application

Problem Description

The objective of this project is to predict the total number of Olympic medals a country can win based on the number of gold, silver, and bronze medals.

Dataset Explanation

The dataset used comes from Kaggle and contains information about countries and their medal counts during the Tokyo 2021 Olympic Games.

Variables Used

  • Independent Variables: Gold, Silver, Bronze
  • Dependent Variable: Total Medals

Data Preparation

The dataset was cleaned and processed by selecting relevant columns and ensuring that all values were numeric. Missing values were checked and handled before training the model.

Regression Graph

The regression graph shows the relationship between the variables and helps visualize how well the model fits the data.

Interpretation of Results

The model indicates a strong relationship between the number of gold, silver, and bronze medals and the total number of medals. As these values increase, the predicted total medals also increase.
