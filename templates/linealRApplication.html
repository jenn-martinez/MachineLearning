Linear Regression - Application

Problem Description

The objective of this project is to predict the total number of Olympic medals a country can win based on the number of gold, silver, and bronze medals.

Dataset Explanation

The dataset used comes from Kaggle and contains information about countries and their medal counts during the Tokyo 2021 Olympic Games.

Data Preparation

The dataset was cleaned and processed by selecting relevant columns and ensuring that all values were numeric. Missing values were checked and handled before training the model.
