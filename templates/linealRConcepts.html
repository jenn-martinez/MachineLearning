Lineal Regression Concepts

What is linear regression?

Linear regression is a supervised learning algorithm used to model the relationship between a dependent variable and one or more independent variables. It is characterized by:

  • It assumes a linear relationship between input and output.
  • It uses a line of best fit to make predictions.
  • It is used for forecasting, trend analysis, and predictive modeling.

Independent and dependent

Dependent variable

The dependent variable is the one whose outcome is conditioned by the independent variable. 7 This variable gives rise to what we want to predict.

It is usually assigned the letter Y as an identifier

Independent variable

The independent variable is the data or factor used to make predictions. It remains constant and impacts the dependent variable.The independent variable is the data or factor used to make predictions. This does not change.

It is usually assigned the letter X as an identifier

Training data

Training data is the information given to the machine learning model so that it learns to make predictions, recognize patterns, and generate content. Training data is typically found in data sheets that contain characteristics describing the data, called attributes. For example, a data sheet on factory equipment includes attributes such as temperature or repair time.

For a model to be considered "well trained", it must have processed a large amount of data and that data must be of high quality.

Regression Line

The regression line is the best-fit straight line that represents the relationship between variables. It minimizes the difference between actual values and predicted values. This line helps us predict the dependent variable for new and unknown data.

Slope and intercept

For simple linear regression, the straight line formula is used.

y= mx + b

The slope (m) indicates how much Y changes when X increases by one unit. The intercept (b) is the value of Y when X = 0.

Assumptions about linear regression

  1. The relationship between X and Y is a straight line
  2. The errors do not affect each other
  3. The error should have the same dispersion among the data
  4. The errors must have a normal distribution

Gradiente descent

It is an optimization technique used to minimize prediction error. It works by repeatedly adjusting parameters to reduce the difference.

How it works

  • Assign random values for the slope and intercept
  • Calculate the error between the predicted and actual values
  • Calculate how much each value contributes to the error
  • Update the parameters to reduce the error
  • Repeat the process until the error is reduced as much as possible.
